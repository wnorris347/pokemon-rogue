This assignment expands upon the previous assignments by implementing the experience and evolution system for battling. Evolution is currently only implemented for Pokemon who evolve by reaching a specific level (no trade evolutions, move-based evolutions, friendship, etc.). As with my version of 1.09, most text that interrupts gameplay can be skipped by pressing any key. NOTE: Although the algorithm uses the Gen IX formula for calculating XP, unlike Generation IX, Pokemon do not gain experience when a wild Pokemon is caught.
